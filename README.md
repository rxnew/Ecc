Ecc
==============
What is this?
--------------
楕円曲線ディフィー-ヘルマン鍵共有 (Elliptic curve Diffie–Hellman key exchange, ECDH) のテストプログラム

===============
How to use?
---------------
cmake をインストールしてください
Ubuntu なら以下のコマンドで入ります
```
$ sudo apt-get install cmake
```

build ディレクトリ内で以下のコマンドを叩いてください
```
$ cmake ..
```

実行
```
$ ./test
```
